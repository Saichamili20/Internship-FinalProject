# 🎬 Movie Sentiment Analysis

## 📌 Project Overview
This project analyzes movie-related text data to understand audience sentiment using data analysis techniques.

The goal is to identify how users feel about movies (positive, negative, or neutral) based on textual data.

---

## 🧠 Objective
- Understand audience reaction to movies
- Identify sentiment patterns in text data
- Extract insights from tags and reviews

---

## 📊 Key Insights
- Most of the reviews show a **positive sentiment**
- Negative sentiment is linked with specific keywords/tags
- Neutral reviews often indicate unclear or mixed opinions

---

## 🛠️ Process Followed
1. Data cleaning and preprocessing
2. Handling missing and duplicate values
3. Exploratory Data Analysis (EDA)
4. Sentiment classification
5. Insight extraction

---

## 💡 Outcome
This analysis helps understand audience perception of movies and can support recommendation systems or content improvement strategies.

## Final Insights
- Most reviews show positive sentiment
- Certain tags influence negative reviews
