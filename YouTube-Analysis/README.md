# 📺 YouTube Viewer Analysis

## 📌 Project Overview
This project analyzes YouTube viewer data to understand engagement patterns and performance trends.

It focuses on how different content types attract viewers.

---

## 🧠 Objective
- Analyze viewer engagement trends
- Identify high-performing content categories
- Understand audience behavior

---

## 📊 Key Insights
- Some content categories consistently get higher engagement
- Viewer activity varies based on content type and timing
- Certain videos have significantly higher reach than others

---

## 🛠️ Process Followed
1. Data collection and cleaning
2. Exploratory Data Analysis (EDA)
3. Visualization of trends
4. Dashboard creation
5. Insight extraction

---

## 💡 Outcome
This analysis helps understand what type of content performs best and can help optimize content strategy for better reach.

## Final Insights
- Engagement varies by content type
- Some categories perform better than others
